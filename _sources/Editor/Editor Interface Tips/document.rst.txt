.. _Editor Interface Tips:

=============================
General Interface Tips
=============================

Fullscreen mode
======================================

To hide taskbar or other operating system interface elements enter Fullscreen Mode by pressing ``[Shift] + [F11]``.

Alternatively, open ``Window`` in menu bar and press ``Enable Fullscreen``

.. figure:: images/07.*


Maximize viewport (Immersive Mode)
======================================

.. figure:: images/04.*

Press ``[F11]`` to switch into immersive mode, quite useful on small screens.

.. figure:: images/05.*

    Press ``[F11]`` to exit or press this button in upper right corner.

.. figure:: images/06.*

You can also enter it using Viewport Options menu located in viewport toolbar.


.. _editor_context_menus:

Context Menus
======================

Quick search inside of context menu
--------------------------------------

After opening context menu, start typing to search it. Use ``[Arrow Keys]`` and ``[Enter]`` to quickly activate functions.

.. figure:: images/01.*
    :align: center
    :width: 80%

Hovering over submenu will search in hovered submenu instead.

.. figure:: images/02.*
    :align: center
    :width: 80%
